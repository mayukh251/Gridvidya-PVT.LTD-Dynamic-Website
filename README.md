# Gridvidya Premium Landing Page

A premium dark futuristic landing page built with **Next.js (App Router)**, **TailwindCSS**, and **Framer Motion** for Gridvidya Pvt. Ltd.

## Install

```bash
npm install
```

## Run (Development)

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Build Production

```bash
npm run build
npm run start
```
